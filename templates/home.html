Personal Learning App

Welcome to your personal spaced repetition learning application. Create new cards with Markdown and LaTeX, then review them to commit them to memory.

Start Reviewing Create a New Card
How It Works
  • Create or Upload Courses: You can write your own courses directly in the editor using Markdown and LaTeX, or upload existing Markdown files.
  • AI-Powered Card Generation: Use AI (Gemini or Anthropic) to automatically generate flashcards from your course content.
  • Free AI Tier: New users can start generating cards right away with a pre-configured API key. You can add your own key for more extensive use.
  • Spaced Repetition: The app uses a spaced repetition algorithm (inspired by SM-2) to schedule card reviews at optimal intervals, maximizing retention.
Telegram Bot Setup
  1. Connect Telegram: Go to Settings and tap Connect Telegram — it opens the bot with a secure link that pairs your account automatically.
